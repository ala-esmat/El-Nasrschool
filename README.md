# 🏫 El-Nasr School Portal

A modern, responsive landing page and information portal designed for **El-Nasr School**. This project focuses on delivering a clean user interface that makes school information accessible and organized.

[**🌐 View Live Demo**](https://ala-esmat.github.io/El-Nasrschool/)

## ✨ Key Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Modern UI:** Clean aesthetic with a focus on readability and school branding.
- **Intuitive Navigation:** Easy access to school news, departments, and contact info.
- **Interactive Elements:** Smooth transitions and hover effects for a professional feel.

## 🛠️ Built With
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and Flexbox/Grid layouts.
- **JavaScript:** For interactive components and navigation logic.

## 📂 Project Structure
- `/css`: Stylesheets and design tokens.
- `/js`: Interactive scripts.
- `/images`: Optimized assets for the school environment.
